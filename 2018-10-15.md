### Dom操作

#### 创建新节点

```
createDocumentFragment();//创建一个dom片段
createElement();//创建一个具体元素
createTextNode();//创建一个文本节点

```

#### 增，删，替换，插
```
appendChild()
removeChild()
replaceChild()
insertBefore()

```

### 查找

```
getElementsByTagName()//通过标签名称
getElementByName()//通过元素的name属性
getElementById()//通过元素id,唯一性

```
